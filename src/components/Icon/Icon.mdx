# Icon

Biblioteca de ícones do Olist Design System. Todos os ícones são SVGs inline com suporte a `currentColor` para herdar cor via CSS.

## Import

```tsx
import { Icon } from '@pedrohenriquevalentim/olist-ds';
```

## Uso básico

```tsx
{/* Tamanho e cor padrão */}
<Icon name="check" size={24} />

{/* Com cor explícita */}
<Icon name="alert-circle" size={20} color="#E64E36" />

{/* Herda cor do elemento pai (ideal em botões e inputs) */}
<Icon name="search" size={16} color="currentColor" />
```

## Nomes disponíveis

Use o sufixo `-fill` para a variante preenchida de cada ícone: `check` → `check-fill`, `alert` → `alert-fill`.
