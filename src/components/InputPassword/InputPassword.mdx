# InputPassword

Campo de senha com toggle de visibilidade. Suporta ícone de lead, tooltip e estados de validação.

## Import

```tsx
import { InputPassword } from '@pedrohenriquevalentim/olist-ds';
```

## Uso básico

```tsx
const [value, setValue] = useState('');

<InputPassword
  label="Senha"
  placeholder="Digite sua senha"
  value={value}
  onChange={setValue}
  hasSupport
  supportText="Mínimo 8 caracteres"
/>

{/* Com ícone de lead */}
<InputPassword
  label="Senha"
  leadIcon={<Icon name="lock" size={16} color="currentColor" />}
  value={value}
  onChange={setValue}
/>
```
