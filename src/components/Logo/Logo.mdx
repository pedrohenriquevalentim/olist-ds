# Logo

Logotipo Olist em três variantes de tamanho. Use `default` em telas principais, `simple` em headers compactos e `symbol` em espaços reduzidos como favicons ou avatares.

## Import

```tsx
import { Logo } from '@pedrohenriquevalentim/olist-ds';
```

## Uso básico

```tsx
{/* Logo completo */}
<Logo size="default" />

{/* Sem círculo laranja, redimensionável */}
<Logo size="simple" />

{/* Apenas o ícone */}
<Logo size="symbol" />
```
