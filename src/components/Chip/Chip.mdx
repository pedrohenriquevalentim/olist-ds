# Chip

Filtro ou tag selecionável. Use em grupos para permitir seleção múltipla de categorias, filtros ou atributos.

## Import

```tsx
import { Chip } from '@pedrohenriquevalentim/olist-ds';
```

## Uso básico

```tsx
const [selecionado, setSelecionado] = useState(false);

<Chip
  label="Eletrônicos"
  isSelected={selecionado}
  onChange={setSelecionado}
/>
```
