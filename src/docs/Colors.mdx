# Cores

Todas as cores são definidas como design tokens no Figma e exportadas via Style Dictionary para `src/generated/variables.css`.

> **Regra:** nunca use valores hex diretamente no código. Use sempre `var(--nome-do-token)`.

export const Swatch = ({ token, hex, label }) => (
  <div style={{ display: 'flex', alignItems: 'center', gap: 12, padding: '6px 0', borderBottom: '1px solid #f0ece0' }}>
    <div style={{ width: 36, height: 36, borderRadius: 6, background: hex, border: '1px solid rgba(0,0,0,0.08)', flexShrink: 0 }} />
    <div style={{ display: 'flex', flexDirection: 'column', gap: 1 }}>
      <code style={{ fontSize: 12, color: '#403f3b' }}>{token}</code>
      <span style={{ fontSize: 11, color: '#827f73' }}>{hex}{label ? ` — ${label}` : ''}</span>
    </div>
  </div>
);

export const PaletteGrid = ({ children }) => (
  <div style={{ display: 'grid', gridTemplateColumns: '1fr 1fr', gap: '0 32px', marginBottom: 32 }}>
    {children}
  </div>
);

## Mapa de status para badges

| Status | Fundo | Texto |
|---|---|---|
| Pendente | `--color-yellow-yellow-50` | `--color-yellow-yellow-600` |
| Aprovado | `--color-blue-blue-50` | `--color-blue-blue-600` |
| Despachado | `--color-green-green-50` | `--color-green-green-600` |
| Cancelado | `--color-red-red-50` | `--color-red-red-600` |
| Em análise | `--color-cyan-cyan-50` | `--color-cyan-cyan-600` |

## Como usar

```css
.elemento {
  /* texto principal */
  color: var(--color-gray-gray-900);

  /* ação primária */
  background: var(--color-blue-blue-500);

  /* erro */
  color: var(--color-red-red-500);

  /* fundo padrão */
  background: var(--color-gray-gray-0);
}
```
